## 1. O'quvchi shaxsiy kabineti (Student Portal)

### 1.1 Login tizimi

Admin har bir talaba uchun **login + parol** yaratadi:
- Login: telefon raqami (yoki admin bergan username)
- Parol: admin tomonidan belgilanadi, talaba o'zgartirishi mumkin
- Kirish sahifasi: `/portal/login` (dashboarddan alohida)
- `Talaba` modeliga `login String? @unique` va `parolHash String?` qo'shiladi

**Sxema o'zgarishi:**
```prisma
model Talaba {
  ...
  login     String?  @unique   // telefon yoki custom login
  parolHash String?            // bcrypt hash
}
```

**Admin tomonidan parol berish:**
- Talaba detail sahifasida "Portal kirish" tugmasi
- Admin parol kiritadi → hash saqlanadi
- Talabaga Telegram orqali login/parol yuboriladi

### 1.3 Texnik implementatsiya

**Middleware:**
```
/portal/* → student session tekshiruvi
/dashboard/* → admin/reception/teacher session tekshiruvi
```

**NextAuth provider qo'shish:**
```typescript
// lib/auth.ts ga qo'shimcha credentials provider
{
  id: "student",
  name: "Talaba",
  credentials: { login, password },
  authorize: async ({ login, password }) => {
    const talaba = await prisma.talaba.findUnique({ where: { login } });
    if (!talaba?.parolHash) return null;
    const ok = await bcrypt.compare(password, talaba.parolHash);
    return ok ? { id: talaba.id, role: "TALABA", ... } : null;
  }
}
```

**Yangi API endpointlar:**
- `GET /api/portal/davomat` — o'z davomati
- `GET /api/portal/jadval` — o'z guruhi jadvali
- `GET /api/portal/tolovlar` — o'z to'lovlari
- `PATCH /api/portal/profil` — parol o'zgartirish

## 2. Moliya moduli (Xarajatlar)

### Sxema:
```prisma
model Xarajat {
  id        String      @id @default(cuid())
  tur       XarajatTur
  summa     Int
  sana      DateTime
  izoh      String?
  createdAt DateTime    @default(now())
}

enum XarajatTur {
  IJARA
  KOMMUNAL
  REKLAMA
  MAOSH
  JIHOZLAR
  BOSHQA
}
```

### Sahifalar:
- `/dashboard/xarajatlar` — CRUD
- `/hisobotlar` ga P&L qo'shish:
  - Daromad (to'lovlar jami)
  - Xarajat (xarajatlar jami)
  - **Foyda = Daromad − Xarajat**

## 4. Avtomatik Telegram Cron

| Hodisa | Vaqt | Xabar |
|--------|------|-------|
| Sinov darsi eslatmasi | Sinov kuni −1, soat 18:00 | "Ertaga sinov darsi..." |
| To'lov eslatmasi | Har oyning 25-i | "Bu oy to'lov qilinmagan..." |
| Tug'ilgan kun | Har kuni 08:00 | "Tug'ilgan kuningiz muborak!" |

**Vercel Cron config (`vercel.json`):**
```json
{
  "crons": [
    { "path": "/api/cron/sinov-eslatma", "schedule": "0 18 * * *" },
    { "path": "/api/cron/tolov-eslatma", "schedule": "0 9 25 * *" },
    { "path": "/api/cron/tugrilgan-kun", "schedule": "0 8 * * *" }
  ]
}
```
